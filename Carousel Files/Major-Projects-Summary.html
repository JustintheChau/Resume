Major Projects

Senior Design Project - Design an Oscilloscope

  • Design a data acquisition system that can measure analog voltage signals and perform basic mathematical operations on analog signals
  • Analog voltage range: -50V to +50V
  • Frequency range: 1Hz to 1MHz
  • Input signal conditioning
  • Analog signal operations: addition, subtraction, differentiation, and integration
  • Digitize signal using an analog-digital converter on a microcontroller
  • Display digitalize signal on a computer using LABVIEW
  • Design system on PCBs using PCB design software
