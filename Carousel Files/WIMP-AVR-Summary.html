Custom Microprocessor Design: WIMP AVR

A Softcore Microprocessor on an FPGA Board Designed using Schematic Capture Files

  • Modeled after AVR Proccessor
  • Custom 16 Bit Instruction Set 16
  • Read/Write capability on FPGA RAM
  • Used as an education tool to discover processor design without knowledge of hardware descriptive languages
  • WIMP AVR report published by ASEE
